About this web app

This project built and trained a CNN model to recover greyscale images to colored images

model

The model used resnet18 as backbone.

It converts RGB images to LAB spaces, and trained only on L space

data model

The user login system is saved as User table (username, password),

The colorized photos are saved as immutable object in AWS S3

The upload history are saved in MongoDB Atlas,

where each colorized photo contains their s3 url and user information

deployment

The app is deployed on AWS EC2 using flask
